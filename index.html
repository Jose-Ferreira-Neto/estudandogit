descobri que quando adiconamos o git add . "TODOS OS ARQUIVOS SÃO ENVIADOS"

Quando damos o commit é para versionar e o push (git push --set-upstream origin master) enviamos para o GITHUB

Quando damos git status vemos como está a atualização dos arquivos

BRANCH

Quando usamos o git branch; avistamos todas as branchs (galhos) do projeto, normalmente separamos para trabalhar sem modificar a produção em real time

git branch -NOME DO BRANCH- ; cria um branch

git checkout -NOME DO BRANCH- ; seleciona o branch para que se possa fazer add; commit e push

MERGE - JUNTAR CÓDIGOS DE DIFERENTES BRANCHS

Primeiro entre na branch principal: git checkout master em seguiada: git merge -NOMA DA BRANCH QUE VOCÊ QUER TRAZER PARA A MASTER- ao final; git push
